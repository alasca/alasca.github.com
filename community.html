You will find here some usefull information about Alasca community.

How to report a bug ?

Simply, go to issues section on GitHub. Feel free to report a new issue or even a new feature.

If you are not sure about your issue, you could send a mail on the users mailing list.

How to stay informed ?

There are several mailing lists you could subscribe to:

  • Announcements only : alasca-announce@googlegroups.com - Low volume
  • General discussions and help : alasca-users@googlegroups.com - Start here!
  • Development and project discussions : alasca-dev@googlegroups.com

There is also the GitHub page where you could follow developments. You could even fork this project, improve it by bringing some new features, and ask for a merge so that your contribution will be introduce into Alasca.

Alasca has a Ohloh page where your could find some code analyses and statistics.
